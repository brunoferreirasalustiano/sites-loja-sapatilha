## Português

### 📝 Sobre o Projeto
A **De Pietri** é uma plataforma de e-commerce minimalista e elegante desenvolvida para a venda de sapatilhas.
O foco do projeto foi criar uma experiência de compra fluida (UX), com um painel administrativo intuitivo para gerenciamento de estoque e 
integração direta com WhatsApp para fechamento de pedidos.

### ✨ Funcionalidades
* **Vitrine Interativa:** Carrossel de produtos com efeito de zoom suave.
* **Carrinho Dinâmico:** Gerenciamento de itens em tempo real com LocalStorage.
* **Checkout Inteligente:** Integração com API ViaCEP para preenchimento automático de endereço.
* **Painel Administrativo:** Área restrita para edição de preços, nomes e troca de fotos em tempo real.
* **Responsividade Total:** Design otimizado para PC, Tablets e Mobile (foco em botões grandes para fácil navegação).
* **Finalização via WhatsApp:** Envio automático dos detalhes do pedido formatados para o vendedor.

### 🚀 Tecnologias Utilizadas
* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (Vanilla).
* **Backend:** Node.js com Express.
* **Integrações:** API ViaCEP.
* **Deploy:** Render & GitHub.

## English

### 📝 About the Project
**De Pietri** is a minimalist and elegant e-commerce platform developed for selling ballet flats (sapatilhas).
The project focus was to create a seamless user experience (UX), featuring an intuitive admin dashboard for inventory
management and direct WhatsApp integration for order fulfillment.

### ✨ Features
* **Interactive Showcase:** Product carousel with smooth zoom effects.
* **Dynamic Cart:** Real-time item management using LocalStorage.
* **Smart Checkout:** Integration with ViaCEP API for automatic address lookup.
* **Admin Dashboard:** Restricted area for updating prices, names, and product photos instantly.
* **Full Responsiveness:** Optimized design for Desktop, Tablets, and Mobile (focusing on large touch-friendly buttons).
* **WhatsApp Checkout:** Automatic sending of formatted order details directly to the seller.

### 🚀 Technologies
* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (Vanilla).
* **Backend:** Node.js with Express.
* **Integrations:** ViaCEP API.
* **Deployment:** Render & GitHub.
